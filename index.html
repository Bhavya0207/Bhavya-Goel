About Me

Hi, my name is Bhavya and I am a student of age, 14. I am currently living and studying in Delhi, India. In fifth grade I had started making animation in a program called scratch. This was my first venture in coding. Soon after, I had joined a computer and robotics academy partnered with Lego where I learned to code a Lego bot using EV3. This was the first time I had started brainstorming to make my own algorithms. After which I changed my focus to actual coding. I started learning Arduino, which was a completely different territory for me as I had jumped from dragging blocks to writing boring text with syntax but eventually I got used to that. Now that I knew a bit or two about hardware I turned to Python. In 2020 when the world was in lockdown, I joined HT Codethon where I dipped my toes into python for the first time. Skip to the present and I am learning competitive python along with web development from the internet. Have a nice day!
